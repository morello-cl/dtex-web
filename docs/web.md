# DTEx® — Landing Page Oficial
## Objetivo del Proyecto

Diseñar y desarrollar una landing page moderna para DTEx®, enfocada en captación comercial, conversión y posicionamiento como plataforma fintech chilena para comercios.

La web debe transmitir:
- Tecnología moderna
- Simplicidad operacional
- Generación de ingresos para el comercio
- Rapidez
- Confianza
- Cercanía con el comercio chileno

DTEx® NO debe verse como una empresa bancaria tradicional.
Debe verse como una startup fintech chilena moderna, cercana y simple.

---

# Mensaje Principal

## Frase principal:
“Convierte tu negocio en un punto de pagos y servicios.”

## Mensajes secundarios:
- Acepta pagos con tarjeta
- Emite boletas electrónicas
- Cobra cuentas y recargas
- Administra turnos y cierres
- Todo desde un equipo Android
- Operado por Sencillito®

---

# Objetivo Comercial de la Web

La web debe:
- Captar comercios interesados
- Generar formularios de contacto
- Incentivar cotización de equipos POS
- Motivar activación de Punto Sencillito®
- Mostrar funcionalidades reales
- Generar confianza tecnológica
- Mostrar que DTEx® ayuda a ganar dinero al comercio

La web debe tener enfoque comercial agresivo, pero elegante y minimalista.
